# 📊 Projeto 03 - Relatório Comercial Interativo

## 📌 Sobre o Projeto

Este projeto foi desenvolvido em Python com o objetivo de simular um sistema de geração de relatórios comerciais.

Diferentemente do Projeto 02, em que os dados eram definidos diretamente no código, nesta versão o usuário informa todas as informações por meio do teclado, tornando a aplicação mais dinâmica e próxima de um cenário real.

Ao final, o programa calcula automaticamente os principais indicadores de desempenho comercial (KPIs) e apresenta um relatório completo na tela.

---

## 🎯 Objetivos

- Praticar interação com o usuário utilizando `input()`;
- Aplicar estruturas condicionais (`if` e `else`);
- Utilizar estruturas de repetição (`while`);
- Trabalhar com operadores matemáticos e relacionais;
- Automatizar o cálculo de indicadores comerciais;
- Organizar o código seguindo boas práticas de programação.

---

## 📊 Indicadores Calculados

- Taxa de Ativação das Agências
- Percentual de Agências com Venda
- Taxa de Conversão
- Ticket Médio
- Percentual de Leads Não Convertidos
- Receita Média por Agência com Venda

## 📈 Contexto

Os dados utilizados são fictícios e têm finalidade exclusivamente educacional.

O projeto foi desenvolvido para praticar Python aplicado à automação de relatórios comerciais e ao cálculo de indicadores utilizados nas áreas de Inteligência Comercial, Business Intelligence e Análise de Dados.
